# 02 C#的System命名空間(namespace)

#### C#程式基礎於.net的框架(framework), 它使用.net中的類別(class)及結構(struct)等資源, 框架內的資源依其用途分存在不同的命名空間中. 

```
     System
        |
        |___ 類別(class)
        |      |___ Console
        |      |___ Object        
        |      |___ String
        |      |___ Math
        |      |___ Array
        |      |___ Enum
        |      |___ Random
        |      |___ Tuple
        |      |___ Exception
        |      |___ ....(略)
        |
        |___ 結構(struct)
               |___ SByte
               |___ Int16        
               |___ Int32        
               |___ Int64
               |___ Byte        
               |___ UInt16 
               |___ UInt32
               |___ UInt64        
               |___ Char 
               |___ Boolean
               |___ Single        
               |___ Double         
               |___ ....(略)
               
                  
         ....(略) 
```
