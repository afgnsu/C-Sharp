# 04 使用簡單型別-01

#### (1) 專案架構 

```
專案
  |__ MainClass.cs
```


#### (2) MainClass.cs

```
using System;

class MainClass 
{
    public static void Main(string[] args)
    {
        int i = 100;
        int j = 200;
        
        Console.WriteLine(i*j);
    }    
}
```

#### (3) 執行結果

```
20000
```


#### (4) 注意事項
```
1. int的儲存極限.
2. 用sbyte, short, int, long儲存.
3. 用byte, ushort, uint, ulong儲存.
4. 型態間的轉換.
```
