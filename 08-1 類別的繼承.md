# 08-1 類別的繼承

#### (1) 專案架構 

```
專案
  |__ Staff.cs 
  |__ Driver.cs  
  |__ MainClass.cs  
專案
  |__ Staff.cs 
  |__ Driver.cs  
  |__ MainClass.cs  
  
  ________________
  | Staff        |
  |______________|
  | + no         |
  | + name       |
  | + rank       |
  |______________|  父親, Base Class
         |
  _______|________
  | Driver       |
  |______________|
  | + license    |
  | + phoneNo    |
  |______________|  兒子, Derived Class    
```


#### (2-1) Staff.cs

```
namespace Ntub.Imd
{
    class Staff
    {
        //------------------------
        // 成員
        //------------------------
        public string no;
        public string name;
        public string rank;
    }
}
```


#### (2-2) Driver.cs

```
namespace Ntub.Imd
{
    class Driver : Staff
    {
        //------------------------
        // 成員
        //------------------------
        public string license;
        public string phoneNo;
    }
}
```



#### (2-3) MainClass.cs

```
using System;
using Ntub.Imd;

class MainClass 
{
    public static void Main(string[] args)
    {
        Driver s = new Driver();
        
        s.no="E001";
        s.name="tommy";
        s.rank="A";
        s.license="AB0001";
        s.phoneNo="0911222333";
        
        Console.WriteLine("no:{0}", s.no);
        Console.WriteLine("name:{0}", s.name);
        Console.WriteLine("rank:{0}", s.rank);
        Console.WriteLine("license:{0}", s.license); 
        Console.WriteLine("phoneNo:{0}", s.phoneNo); 
    }    
}
```

#### (3) 執行結果

```
no:E001
name:tommy
rank:A
license:AB0001
phoneNo:0911222333
```


#### (4) 注意事項
```
1. 注意繼承的寫法.
2. 並非父親的成員或方法都可由兒子取用, 如由private修飾者, 即使是兒子亦無法取用.
```
