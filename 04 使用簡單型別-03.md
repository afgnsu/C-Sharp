# 04 使用簡單型別-03

#### (1) 專案架構 

```
專案
  |__ MainClass.cs
```


#### (2) MainClass.cs

```
using System;

class MainClass 
{
    public static void Main(string[] args)
    {
        char i = 'A';
        byte k = (byte)i;

        Console.WriteLine(i);
        Console.WriteLine(k);
    }    
}
```

#### (3) 執行結果

```
A
65
```


#### (4) 注意事項
```
1. 字元可以存中文.
2. 字元用單引號, 字串用雙引號.
```
