# 08-2 有建構元的父親

#### (1) 專案架構 

```
專案
  |__ Staff.cs 
  |__ Driver.cs  
  |__ MainClass.cs  
```


#### (2-1) Staff.cs

```
namespace Ntub.Imd
{
    class Staff
    {
        //------------------------
        // 成員
        //------------------------
        public string no;
        public string name;
        public string rank;
        
        //------------------------
        // 建構元
        //------------------------
        public Staff(string no, string name, string rank)
        {
            this.no = no;
            this.name = name;
            this.rank = rank;
        }
    }
}
```


#### (2-2) Driver.cs

```
namespace Ntub.Imd
{
    class Driver : Staff
    {
        //------------------------
        // 成員
        //------------------------
        public string license;
        public string phoneNo;
        
        
        //------------------------
        // 建構元
        //------------------------
        public Driver(string no, string name, string rank, string license, string phoneNo) : base(no, name, rank)
        {
            this.license = license;
            this.phoneNo = phoneNo;
        }        
    }
}
```



#### (2-3) MainClass.cs

```
using System;
using Ntub.Imd;

class MainClass 
{
    public static void Main(string[] args)
    {
        Driver s = new Driver("E001", "tommy", "A", "AB0001", "0911222333");
        
        Console.WriteLine("no:{0}", s.no);
        Console.WriteLine("name:{0}", s.name);
        Console.WriteLine("rank:{0}", s.rank);
        Console.WriteLine("license:{0}", s.license); 
        Console.WriteLine("phoneNo:{0}", s.phoneNo); 
    }    
}
```

#### (3) 執行結果

```
no:E001
name:tommy
rank:A
license:AB0001
phoneNo:0911222333
```


#### (4) 注意事項
```
1. 注意呼叫父親建構元的寫法.
2. 如果寫了有參數的建構元, 如果不傳參數, 也要自己再寫無參數的建構元.
```
