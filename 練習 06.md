### 2.MainClass.cs

```
using System;
using System.IO;

class MainClass {
    public static void Main (string[] args) {
        //------------------
        // 輸入檔
        //------------------
        StreamReader file = new StreamReader("data.csv");         
        
        //----------------------------
        // 依序讀入各行資料
        //----------------------------
        string line;
        
        while((line = file.ReadLine()) != null)  
        {  
            // 將讀入整行以逗號分割  
            string[] items = line.Split(',');
            
            // 取出序號及職等及銷售金額
            int serNo = Int32.Parse(items[0]);
            char rank = Char.Parse(items[1]);
            int total = Int32.Parse(items[2]);


            // 增加程式內容, 計算傭金
            int commission = 0;
            
            // 輸出
            Console.WriteLine("{0}  \trank:{1} \ttotal:{2}  \tcommission:{3}", serNo, rank, total, commission);
        }  

        //----------------------------
        // 關檔
        //----------------------------       
        file.Close();  
    }
}
```

#### 依以下規則計算每種銷售金額的傭金.

```
以下是傭金計算方式(小數捨去):
----------------------------------------
職等      傭金比
----------------------------------------
 A        超過20萬, 傭金18%; 否則15%.
 B        超過15萬, 傭金13%; 否則11%.
 C        超過10萬, 傭金8%; 否則6%.
其他      超過8萬,  傭金5%; 否則3%.    	 
----------------------------------------
```
