### 2.MainClass.cs

```
using System;
using System.IO;

class MainClass {
    public static void Main (string[] args) {
        //------------------
        // 輸入檔
        //------------------
        StreamReader file = new StreamReader("data.csv");         
        
        //----------------------------
        // 依序讀入各行資料
        //----------------------------
        string line;
        
        while((line = file.ReadLine()) != null)  
        {  
            // 將讀入整行以逗號分割  
            string[] items = line.Split(',');
            
            // 取出序號及用電度數
            int serNo = Int32.Parse(items[0]);
            int degree = Int32.Parse(items[1]);


            // 增加程式內容, 計算電費
            int fee = 0;
            
            // 輸出
            Console.WriteLine("{0}    \tdegree:{1}    \tfee:{2}", serNo, degree, fee);
        }  

        //----------------------------
        // 關檔
        //----------------------------       
        file.Close();  
    }
}
```

### 4.問題

#### 依以下規則計算每種用電度數的電費.

```
以下是台電電費計算方式(小數捨去):

---------------------------------------
「非營業用電」   夏月每度（元） 
---------------------------------------
120度(含)以下部分   2.10      	 
121-330度部分       3.02      	 
331-500度部分       4.39      
501-700度部分       5.44      	 
701-1000度部分      6.16      	 
1001度以上部分      6.71      	  
---------------------------------------

假設用電1200度, 電費為:
2.10*120 + 3.02*210 + 4.39*170 + 5.44*200 + 6.16*300 + 6.71*200
= 5,910元
```
