# 06-2 結構中的方法

#### (1) 專案架構 

```
專案
  |__ Student.cs  
  |__ MainClass.cs  
```


#### (2-1) Student.cs

```
using System;

namespace Ntub.Imd
{
    public struct Student
    {
        //---------------------------------------------        
        private string name;
        private char gender;
        private int age;
        
        //---------------------------------------------
        public void setData(string name, char gender, int age)
        {
            this.name=name;
            this.gender=gender;
            this.age=age;
        }

        //---------------------------------------------        
        public void printData()
        {
            Console.WriteLine("name:{0}", name);
            Console.WriteLine("gender:{0}", gender);
            Console.WriteLine("age:{0}", age);
        }
        //---------------------------------------------           
    }
}
```


#### (2-2) MainClass.cs

```
using Ntub.Imd;

class MainClass 
{
    public static void Main(string[] args)
    {
        Student s1 = new Student();
        s1.setData("Tommy", 'M', 18);
        s1.printData();
    }    
}
```

#### (3) 執行結果

```
name:Tommy
gender:M
age:18
```


#### (4) 注意事項
```
1. 用new指令建立結構物件時, 該物件中的欄位(name, gender, age)均會賦予初值; 欄位均有值時, 該結構物件才可使用.
2. 結構是一種實質型態. 
3. 雖然和class很像, 但是結構不支援繼承.
```
