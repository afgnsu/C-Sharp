# 01 C# 的型別(Types)

## 值型別(Value Types)和參考型別(Reference Types)的略圖

```
  型別(Types)
        |
        |___ 值型別(Value Types)
        |         |
        |         |___ <<基本型別>>
        |         |       |
        |         |       |___ <數字>
        |         |       |      |__ <整數>
        |         |       |      |     |__ sbyte,  short,   int,   long
        |         |       |      |     |__ byte,   ushort,  uint,  ulong                  
        |         |       |      |
        |         |       |      |__ <浮點數>
        |         |       |            |__ float,  double,  decimal
        |         |       |   
        |         |       |___ <字元>
        |         |       |      |__ char
        |         |       |   
        |         |       |___ <布林>
        |         |              |__ bool    
        |         | 
        |         |___ <<結構>>  
        |         |       |__ 內建 struct  
        |         |       |__ 自建 struct         
        |         |
        |         |___ <<列舉>>  
        |                 |__ 內建 enum 
        |                 |__ 自建 enum         
        |
        |        
        |___ 參考型別(Reference Types)
                  |
                  |___ <<類別>>
                  |       |___ object
                  |       |___ string 或 System.String
                  |       |___ 內建 class  
                  |       |___ 自建 class     
                  |
                  |___ <<介面>>
                  |       |___ 內建 interface  
                  |       |___ 自建 interface
                  |
                  |___ <<陣列>>
                  |       |___ System.Array 
                  |
                  
         ....(略) 
```
