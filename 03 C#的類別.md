# 03 C#的類別(class)

#### .net的框架提供許多內建類別, 它們被分類儲存在不同的命名空間中; 而使用者也可依需要自建客制化類別. 一個類別包括屬性(properties)及方法(methods), 屬性是資料, 方法是函式. 屬性及方法都可依需求設計靜態(static)內容, 而方法中有一些被設計為建構元(constructors). 

```
     類別(class)
        |
        |___ 屬性(properties), 包括數字, 布林, 字元, string, 或是其他結構及類別等的宣告, 用來儲存資料.
        |      |___ 非靜態屬性 (類別生成的每個物件各有一份)
        |      |___ 靜態屬性   (依附在類別中的變數, 只有一份, 可用 <類別名稱.靜態屬性名稱> 直接取用)
        |
        |___ 方法(methods), 用來操作類別中的屬性及傳入參數的函式.
               |___ 建構元     (類別生成物件後自動被執行的函式)
               |___ 非靜態方法 (操作在特定的物件上, 用 <物件名稱.方法名稱> 呼叫)     
               |___ 靜態方法   (依附在類別中的函式, 用 <類別名稱.方法名稱> 呼叫)     
                               
         ....(略) 
```


#### 以System中的String類別為例

```
     String類別
        |
        |___ 屬性
        |      |___ 非靜態屬性        
        |      |       |___ Chars[Int32]  (存放字串內容)
        |      |       |___ Length        (存放字串長度)
        |      |
        |      |___ 靜態屬性        
        |              |___ (無)     
        |
        |___ 方法
               |___ 建構元
               |      |__ String(Char*)
               |      |__ String(Char*, Int32, Int32)
               |      |__ String(Char, Int32)
               |      |__ ...(略)               
               | 
               |___ 非靜態方法  
               |      |__ Clone()
               |      |__ CompareTo(Object)
               |      |__ Equals(Object)
               |      |__ ...(略)                  
               |
               |___ 靜態方法
                      |__ Compare(String, Int32, String, Int32, Int32)
                      |__ Concat(Object, Object)
                      |__ IsNullOrEmpty(String)
                      |__ ...(略)       
                               
         ....(略) 
```
